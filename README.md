# Heyfaaz Stopwatch ⏱️

A simple, clean, and fully functional **stopwatch** built with **HTML, CSS, and JavaScript**.  
This project runs entirely in the browser and can be used for timing tasks, exercises, or just for fun.

---

## Features

- Start, Stop, and Reset the timer
- Shows **minutes, seconds, and milliseconds**
- Stylish and responsive design
- Works on desktop and mobile browsers
- Easy to extend with features like laps or sounds
